# PointLightController スクリプト README

## 概要
このスクリプトは、Unityのゲーム開発においてポイントライトの色を切り替えるためのものです。プレイヤーが特定のキー（Fキー）を押すことで、ライトの色が赤色とエメラルドグリーンの間で切り替わります。また、色の変更に応じてグローバル変数`GlobalVariables.isFirst`の値も更新されます。


（重要）Second,Thirdも同様です。



## スクリプトの内容

### プライベート変数
- `pointLight`: ポイントライトコンポーネントを保持するための変数
- `isRed`: 現在のライトの色が赤色かどうかを示すブール値（初期値は`true`）

### Unityのライフサイクルメソッド

#### Start
```csharp
void Start()
```
スクリプトが有効になったときに呼び出されます。ポイントライトコンポーネントを取得し、初期色として赤色を設定します。ポイントライトコンポーネントが見つからない場合はエラーメッセージを表示します。

#### Update
```csharp
void Update()
```
毎フレーム呼び出されます。プレイヤーがFキーを押した場合に、`isRed`の値を反転させてライトの色を更新します。

### プライベートメソッド

#### UpdateLightColor
```csharp
void UpdateLightColor()
```
`isRed`の値に応じてライトの色を変更します。赤色の場合は`GlobalVariables.isFirst`を`false`に、エメラルドグリーンの場合は`true`に設定します。

## 使用方法

1. **スクリプトの配置**
   このスクリプトをポイントライトを持つGameObjectにアタッチしてください。ポイントライトコンポーネントがアタッチされていることを確認してください。

2. **ライトの色の切り替え**
   ゲーム実行中にFキーを押すことで、ライトの色が赤色とエメラルドグリーンの間で切り替わります。初期設定ではライトの色は赤色です。

3. **グローバル変数の更新**
   ライトの色が赤色に変更された場合、`GlobalVariables.isFirst`は`false`に設定されます。エメラルドグリーンに変更された場合は`GlobalVariables.isFirst`は`true`に設定されます。これにより、他のスクリプトやゲームロジックでライトの色に応じた処理を行うことができます。

## 注意点
- ポイントライトコンポーネントがアタッチされていない場合、エラーメッセージが表示され、スクリプトは正常に動作しません。
- ゲームの進行状況や他の条件に応じて、`GlobalVariables`クラスの変数を適切に設定してください。

このスクリプトを使用することで、ゲーム内でのライトの色の切り替えと、それに伴うゲームロジックの変更を簡単に実装することができます。
